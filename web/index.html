DryFoodie

Perangkat pintar yang mampu mendeteksi jumlah makanan kering yang tersisa di dalam storage. Data ini akan ditampilkan secara real-time pada layar LCD dan juga dapat diakses melalui web client. Selain itu, web client juga dapat digunakan untuk menjatuhkan makanan secara otomatis, memastikan kucing Anda selalu memiliki makanan yang cukup.
